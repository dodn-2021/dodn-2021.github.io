简单易懂的 pxx 量度表

※ 量度表只是 pxx 的无处断裂的反面表现.

cccp
  • 新的意识形态建筑直接在过去的废墟上拔地而起. 人们经历战争、冒险和权力斗争, 为过去的故事建造现代的舞台.
  • 这不是因为我不想这么做: 只是因为没有余力想象更多. 共产主义指示我们暂时抛弃阶级矛盾, 甚至在明年它就要建成——那么为什么还要质疑那些住在苏维埃宫 (原基督救世主大教堂) 中的首脑呢?
  • “极左” 的极限
scc
  • 人们厌倦了权力斗争, 但是将它挪作为自己谋利的工具: 所谓哲人说得好, 在这里做一个收租人是最偷着乐的事情.
  • 投资过度、引导过度、输出过度. 所有的生活都建立在过度的不平等和各种输出上: 情绪输出、文化输出、人口输出. 我们生活在 scc 的红旗下, 下一步将之若从? 没有人能给出答案.
  • 我们的真实生活
mm
  • 我们容许一切, 包括死亡. 有人说过, 无政府主义只不过是第一国际走错了方向. 那么为何不纠正它? 凭借机械的个人的神话故事让所有人暂时地感受永恒实践的力量.
  • 技术、传说和叙事——上至一切只是为了让时间符号化. 人的生活可以被支付, 当然也可以被购买——当然你要凭借自己出生所遗传的社会基因与个人优势. 感受先于一切, 伟大的自由公民们!
  • 永久革命
mm / pxx
  • 在死亡之前一切都是被允许的. 我们曾经目睹一切: 那是机械母亲所传递给青年公民的殷切教诲. 我们在得到适当的教育之后, 将会以自己的力量改造一切.
  • 性滥交、反文化和不纯粹的解放人类. 谁又能说我们不是追求这样的一生呢? 但大家都知道分寸在哪里: 除非你没有上过学校, 或者已经无惧死亡. 唯一的问题是: 这里应该存在一个永恒的领导者吗?
  • 技术永生
mm // pxx
  • 人们在时间中选择自己的身体和生活. 当 mm 的技术卸除自己的伪装, 我们看见的就只是两派科学家为了自己的所谓理论争夺人类未来——这绝对不被允许.
  • 无论你是想要做什么——这里没有边界, 但也没有指导. 人们在和巨大的自动障碍的交互中学习如何达到自己的目的, 而这种结果并不直接作用于社会: 集体意志终究要通过机器和无数技术员合力计算出来.
  • 辅助进化
mm x pxx = (mm) // (mm / pxx) // (pxx)
  • 死亡不存在或者 “你” 不存在——选一个吧. 是想要在幕后管理所有人的生活, 还是在台前仅仅专注于人类的进步? 当然 [记忆消除] 是不存在的.
  • 当然这个选择只是很久、很久以前才有人定下的了. 身份或许真的像男女一样——只不过它的界线已经十分模糊, 没有人会真的在意. 总之, 为了这样一个选择仍然存在 (并且保住自己的), 我们需要 mm.
  • 互联网
pmm
  • 准确来说, pmm 指的是 mm // pmm 这个关系本身; 相应地, 此时 mm 就指的是 mm / pmm 本身. 当然没有人会喜欢理论, 那么 pmm 的真正意思是: 我们不太喜欢技术.
  • 我们已经目睹历史的进程: 任何一个意识形态的背后总是有无数的谎言. 现在技术用来掩盖 (甚至是实现) 这些操纵——人毕竟还是要生活的, 无论如何, 不是吗?
  • (后) 启示录
pmm / scc
  • 物质交易和利益输送, 即使它们要发生也应该在系统中被所有人了解. 我们知道这会让社会变成许多微小的团体: 每个人都没有只是为了它们自己而阻止群体逐利的权利.
  • 计划经济用来指导人们的市场竞争. 我们在有限的经济产出中加入了精神激励和游戏一般的上瘾机制——你们有听说过那个 CODE135 rank1 的团队成员在昨天就换了两家研究队吗?
  • 工团主义
pmm // scc
  • 创造物质价值的应该是所有人的真诚劳动. (全民的) 生活有了方向, 生命才有力量. 用劳动换取自己的价值, 只有通过劳动才能实现自己本身——无论是脑力劳动还是体力劳动.
  • 难道没有人跟你说过, 劳动、生活和家庭, 这是一件很简单的事情吗? 我们知道在这之上有着无数的理想, 有着敌人和战友、革命和捍卫革命. 那么就用切实的成绩证明, 我们能做到!
  • 计划经济
pmm x scc = (pmm) // (pmm / scc) // (scc)
  • 末世中也要生活, 我们必须承认这是一切的结束, 并且再也不会有人类了! 在这里, 我们唯一能做的就是留下对世界残余最好的印象, 并且让——即使是遥远的未来——人类文明保持完整.
  • 即使这个末世的期限是永久. 打开收音机, 播放从货架上清理的唱片. 前核爆的故事仍然在现在被引用, 只不过每一次都激起人们不同的情感. 为了生存, 以及自己的伤口.
  • 安资民布
pxx
  • UN 的意思就是 USA / NAZI. 别惊慌, 我的意思是, 即使是和纳粹打冷战, 那个资本主义帝国都几乎必定会赢. 现在它赢了, 尽管我不知道下一次是不是这样.
  • 因为我们是一个拥有数千年文明、十数亿人口的国家? 别扯了. 这跟你没有一点关系. 让我猜猜, 你能做的就只有在一旁感受被抛弃后的自由. 只不过现在它属于所有人.
  • 流动边缘
farm
  • 新的科学技术直接在人们的原始欲望中发展出来. 似乎只是一瞬间——人们对财富、权力和扭曲精神的满足指引我们走上这条螺旋上升的道路.
  • 当然了, 首先是要喂饱所有人. 并且让人们——适当地——供应不足. 比如说, 设想一下即使是你坐在电脑前看着 pornhub, 仍然是需要吃喝的, 当然这不可能免费提供——并且你也不能永久射精.
  • “极右” 的极限
cccp x farm = pxx x pxx
  • 一切都取决于观察屏幕的方法: 我们没办法用像素点、扫描线或染料来概括这种展出艺术, 血迹、CENSORED, 和内官脏器都仅仅作为流派, 至于它同样是个工作——观众从非线性叙事中就已满足.
  • 它从未改变, pxx 无意在度量表中找到一个最 “好” 的位置组合. 它只需要永远作为一个选项存在, 就总会有傻子想起原来还有这么一道方便的选择题.
